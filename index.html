  • clearfix的使用（父元素不设置高）

    当A包B、C两个盒子，B和C都有浮动，A的高度就变成0，这时给A加clearfix，A就有高度了。
    第一种 /* .clearfix:before, .clearfix:after { content: '\20'; display: table; height: 0; overflow: hidden; } .clearfix:after { clear: both; } */
    第二种 /* .clearfix:before,.clearfix:after{ content:""; display:table; } .clearfix:after{clear:both;} .clearfix{ *zoom:1;IE/7/6 } */
    第三种 .clearfix:after { content:"\200B"; display:block; height:0; clear:both; } .clearfix {*zoom:1;}/*IE/7/6*/
